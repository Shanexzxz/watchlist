# Watchlist · Watch List 项目

> 投资并购部 · 每日 Watch List（多行业组）
> 完整产品需求见 [`PRD-内容金融组-WatchList.md`](./PRD-内容金融组-WatchList.md)

## 用途

每交易日展示各行业组关注标的的当日行情 + 持仓信息。

- **Phase 1（当前）**：PC 端 Web 页面，支持多组切换、搜索、列排序。
- **Phase 2（后续）**：邮件自动发送，作为 Web 的补充通道。

## 文件结构

| 文件 | 用途 |
|---|---|
| `PRD-内容金融组-WatchList.md` | **产品需求（源头）** — 字段定义、视觉规范、数据源、权限模型 |
| `watchlist-web.html` | **PC 端 Web 页面** — 独立页面，含 Tab 切换/搜索筛选/列排序，与邮件版独立互不干扰 |
| `content-finance-daily.html` | 邮件版视觉/数据 demo — 前端邮件样式对齐此文件 |
| `generate.py` | Demo 生成脚本（仅供产品经理做原型用，工程化时重写） |

## 页面对比

| 维度 | `watchlist-web.html`（PC 端） | `content-finance-daily.html`（邮件版） |
|---|---|---|
| 用途 | 浏览器交互查看 | 邮件只读快照 |
| 交互 | Tab 切换 / 搜索过滤 / 行业筛选 / 列排序 | 无交互 |
| 顶栏 / Tab / 工具栏 | 有 | 无 |
| 数据源 | 同（Bloomberg + TIM） | 同 |
| 表格样式 | 蓝白交替行 / 黑色边框 / 涨绿跌红 | 完全一致 |

## 视觉对齐

表格区域样式 **必须严格遵循** `content-finance-daily.html` 的呈现：
- 蓝白交替行（`#fff` / `#C6D9F1`）
- 黑色 1px 细边框
- 涨绿 `#006400` / 跌红 `#c00`
- 金色 `*` 标记 TPP 持仓

PC 端新增的顶栏、Tab 栏、工具栏等交互元素，视觉风格应与表格区域保持一致（白色背景、蓝色 `#185FA5` 主色、浅灰边框）。

任何视觉调整先对比 demo。
