# Watchlist · 邮件项目 PRD

> 投资并购部 · 每日 Watch list 邮件  
> 最后更新：2026-06-16

## 背景

每个投资组维护自己的 watch list，每日向组内成员推送一份统一格式的 HTML 邮件，覆盖「昨日行情 + 当前持仓」两类信息。

## 数据

### 行情（T-1 收盘）

数据源唯一为 Bloomberg；拉不到一律显示 `-`。

- 收盘价（原币种，2 位小数）
- 涨跌幅、YTD / 30 天 / 365 天涨跌幅
- 交易额
- 市值（USD M）、Adjustment PE (2026E)

### 持仓（TIM，组内 Excel 维护）

- MA 持有价值（USD M）、MoC
- 是否在模拟仓、TPP 持仓

## 输出

一份 HTML 邮件，结构如下：

1. **标题**：`【{组名}】Watch list on {YYYY-MM-DD}`
2. **顶部摘要**：关键变化（如市值/涨跌超阈值名单、未来业绩公布日等）
3. **指数表**：4 只 FinTech 主题指数
4. **行业分组表**：按子行业（如电子银行和钱包、信贷、券商和交易所、支付、金融软件、保险、电商金融平台、加密货币、内容娱乐等）分组展示标的，每只一行
5. **脚注**：TPP 持仓等特殊标记说明

> 邮件样式**严格遵循 demo 模板**（`content-finance-daily.html`），任何调整先比对 demo。

## 频率

每个交易日收盘后（约次日 8:00）生成并发送。
