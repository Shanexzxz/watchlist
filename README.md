# Watchlist · 邮件项目 PRD

> 投资并购部 · 内容金融组每日 Watch list  
> 最后更新：2026-06-16

## 目标

按「组」为单位，每个投资组维护自己的 watch list，每日定时拉取行情信息 + 持仓信息（TIM），生成符合统一规范的 HTML 邮件，发送给组内成员。

---

## 数据维度

### 1. 行情信息（T-1 收盘数据）

> 唯一数据源：**Bloomberg**。拉不到的数据统一显示 `-`，不留空。

| 字段 | 格式 | 示例 |
|---|---|---|
| 收盘价 | 原币种 + 2 位小数 | `HKD 23.46` / `USD 97.54` |
| 涨跌幅 | 带符号 % | `+1.6%` / `-2.7%` |
| 交易额 | 原币种 + 千分符 | `1,039,816`（单位 USD'000） |
| Adjustment PE (2026E) | 数值 | `-` |
| 市值 | USD M | `-` |
| YTD 涨跌幅 | 带符号 % | `+13%` |
| 30 天涨跌幅 | 带符号 % | `+22%` |
| 365 天涨跌幅 | 带符号 % | `-23%` |

### 2. 持仓信息（TIM）

> 来源：组内 Excel watchlist 模板（每组一份）。

| 字段 | 说明 |
|---|---|
| MA 持有价值 | 单位 USD M，2 位小数 |
| MoC | Multiple on Cost，2 位小数 + `x` |
| 是否在模拟仓 | `Y` / `N` / `-` |
| TPP 持仓 | `Y` / `N` / `-`（标注于公司名后 `*`） |

## 邮件结构

```
┌─────────────────────────────────────────────┐
│ 【{组名}】Watch list on {YYYY-MM-DD}          │  标题
│ {数据说明/免责声明}                            │  灰色斜体小标
├─────────────────────────────────────────────┤
│ • Market cap over USD10B ...                 │  顶部摘要
│ • Market cap under USD10B ...                │  （可多个 bullet）
│ • Upcoming Earnings ...                      │
├─────────────────────────────────────────────┤
│ 指数表（10 列：指数 | Date | Change | ... ）   │  行情数据
├─────────────────────────────────────────────┤
│ {行业1}（12 列：公司 | Date | Change | ...）   │  按行业分组
│ {行业2}                                        │
│ ...                                          │
├─────────────────────────────────────────────┤
│ 备注：* 为TPP有持仓公司（{list}）              │  脚注
└─────────────────────────────────────────────┘
```

### 行业表 12 列结构

| # | 列 | 来源 |
|---|---|---|
| 1 | 公司 | TIM |
| 2 | Date | 行情 |
| 3 | Change | 行情 |
| 4 | Turnover (USD'000) | 行情 |
| 5 | Adj. P/E (2026E) | 行情 |
| 6 | Mkt Cap (USD M) | 行情 |
| 7 | Price Change (YTD) | 行情 |
| 8 | Price Change (L30D) | 行情 |
| 9 | Price Change (L365D) | 行情 |
| 10 | MA 持有价值 (USD M) | TIM |
| 11 | MoC | TIM |
| 12 | 模拟仓 | TIM |

> 指数表只有前 9 列（无 MA/MoC/模拟仓）。

## 视觉规范（严格遵循 demo）

- **行底色**：奇数 `#fff`，偶数 `#C6D9F1`（蓝白交替）
- **表头**：`#e8e8e8` 灰底
- **边框**：1px `#000`
- **涨绿 `#006400`，跌红 `#c00`**
- **TPP `*` 角标**：金色 `#B8860B` 加粗
- 邮件 HTML 模板参考：`content-finance-daily.html`

> 任何视觉调整需先对比 demo 样式，保持一致。

---

## 数据流

```
Excel watchlist ──┐
                  ├──> generate.py ──> content-finance-daily.html ──> 邮件发送
Bloomberg API  ──┘
```

- **Bloomberg 拉数**：每天 T 日收盘后（约 T+1 早上 8:00）拉取
- **Excel 更新**：组内维护，每周/每月更新一次
- **HTML 生成**：`python3 generate.py`
- **发送**：手动 / 接入 SMTP 自动化
