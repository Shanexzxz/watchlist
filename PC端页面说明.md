# PC 端 Watch List 页面说明

> 描述 PC 端页面的功能与设计要求。
> 最后更新：2026-06-22

---

## 1. 页面用途

供投资并购部 8 个行业组成员在 PC 浏览器查看各自组的 Watch List 行情数据。每个用户默认只看本组，部分用户可跨组查看（通过 Tab 切换）。

**与邮件版的关系**：PC 端页面和邮件版页面完全独立，互不干扰。邮件版为只读快照，PC 端为交互页面。

---

## 2. 页面结构

```
顶栏     Watch List logo | 用户名 · 可访问 X/8 组
Tab 栏   [内容金融组] [消费零售组] ... （仅显示有权限的组）
工具栏   T-1 日期标签（右对齐）
主内容   标题 → 摘要 bullets → 指数表 → N 个行业表 → 脚注
```

---

## 3. 交互功能

| 功能 | 说明 |
|---|---|
| Tab 切换 | 点击切换行业组，内容整体更新 |

---

## 4. 核心设计：各组配置独立

**这是最重要的设计原则。**

三件事各组独立、互不影响：
1. **行情指标独立** — 各组订阅的行情指标可能不同，某组增减指标列不影响其他组
2. **列宽/表头独立** — 列数、列宽、列标题由各组自行定义
3. **标的清单独立** — 各组维护自己的指数列表和行业/标的清单

---

## 5. 管理后台

与查看页面完全独立，用于维护每个组的 Watch List 配置。

### 5.1 功能

| 功能 | 说明 |
|---|---|
| Tab 切换 | 8 个行业组，切换后管理该组配置 |
| 指数管理 | 添加 / 删除 / 拖拽排序 / 编辑指数名称和 BBG 代码 |
| 行业分类管理 | 添加 / 删除 / 拖拽排序 / 编辑行业名称 |
| 标的管理 | 每个行业下添加 / 删除 / 拖拽排序 / 编辑标的（公司名、BBG 代码、MA 持有价值、MoC、TPP 持仓、模拟仓） |
| Excel 导出 | 将当前组数据导出为 `.xlsx`，格式与 `demo-内容金融.xlsx` 一致 |
| Excel 导入 | 读取 `.xlsx` 文件，覆盖当前组数据 |
| 重置为默认值 | 恢复为初始数据（仅内容金融组有预填数据） |

### 5.2 拖拽排序

排序交互为拖拽，三处可拖拽：
- **指数列表** — 拖拽指数项排序
- **行业卡片** — 拖拽卡片排序行业
- **标的行** — 拖拽行排序标的

### 5.3 Excel 模板格式

与 `demo-内容金融.xlsx` 一致：

```
【指数】
         BBG代码 | 指数名称
【行业名】
         BBG代码 | 公司名称 | MA持有价值 | MoC | TPP持仓 | 模拟仓
```

- `【指数】` 段：每行一个指数，B 列为 BBG 代码，C 列为指数名称
- `【行业名】` 段：B 列 BBG 代码、C 列公司名称、D 列 MA 持有价值、E 列 MoC、F 列 TPP 持仓（Y/N）、G 列模拟仓（Y/N）
- 段以 A 列的 `【xxx】` 标记分隔
